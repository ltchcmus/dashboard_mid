# Data Folder

This folder stores all datasets and generated artifacts in clear stages.

## Subfolders

- `raw/`: Source files copied directly from origin systems.
- `interim/`: Cleaned or joined data during transformation.
- `processed/`: Final tables used by dashboard and generation modules.
- `outputs/`: Generated presentation artifacts.

## Output conventions

- Slide markdown drafts: `outputs/slides/markdown/`
- Slide decks (PPTX): `outputs/slides/pptx/`
- Per-speaker scripts: `outputs/scripts/individual/`
- Final merged scripts: `outputs/scripts/final/`
